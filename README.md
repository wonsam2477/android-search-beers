# Punk API Test
맥주검색 API를 사용하여 맥주검색 및 상세정보 기능이 구현된 안드로이드 데모

## 사용 언어

- Kotlin (100%)

## 앱 구성

- MVVM + AAC + LiveData + Rx

## 주 사용 라이브러리

### Networking
- Retrofit2

### Image Loader
- Glide

### Rx
- RxJava3, RxAndroid3, RxBinding4

### UI Components
- Paging3, Navigation

### Dependency Injection
- Dagger Hilt
